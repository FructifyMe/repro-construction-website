## Outstanding asks for Mark (likely the richest source)

- **CompanyCam account access.** Several `2023/07/aHR0cHM6...jpeg` filenames on the site are base64-encoded CompanyCam exports, which strongly implies Repro has an active CompanyCam workspace. CompanyCam stores every site photo every project executive shoots, organized by project. **This is almost certainly a 10×–100× larger photo library than what's been published to the site.** Worth one ask.
- **Square footage and dollar value** for each flagship project. None of this is on the public site. Needed for fact-dense case study pages (per GEO supplement).
- **Franchisor sign-off** to feature each named QSR brand by name on the new site. Some franchise agreements restrict GC publicity. Dunkin', Jersey Mike's, Dave's, Wayback — each may need a one-line approval.
- **Newer flagship photos.** Public site hasn't been updated since Feb 2024. If Repro completed notable projects in 2024–2026, those aren't reflected anywhere publicly yet.
